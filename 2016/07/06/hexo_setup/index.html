最快方法搭建Hexo博客

注：此方法指的是在Windows下。已经在Windows 10 64bit下实测。

  1. 安装node.js
    此软件可以在 https://nodejs.org/en/ 下到最新版
  2. 安装git
    此软件可以在 https://git-for-windows.github.io/ 下到最新版。
  3. 运行安装命令
    最新版的安装命令来自于hexo官网。
    创建一个用安装hexo的目录，如D:\hexo
    用管理员权限打开cmd,转到刚才建立的那个目录。
    依次运行以下命令
    npm install hexo-cli -g
    hexo init blog
    cd blog
    npm install
    hexo server

博客在本地即可运行。
